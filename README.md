# Monitoramento de Cotação do Dólar

## Descrição
Este projeto consiste em um script desenvolvido em Python para automatizar a coleta e o monitoramento da cotação do dólar. O objetivo principal é realizar o consumo de dados financeiros via API e estruturar essas informações para consultas rápidas.

## Tecnologias Utilizadas
* **Python 3.14.4**: Linguagem base do projeto.
* **Requests**: Responsável pelo consumo da API de cotações financeiras.
* **JSON**: Formato utilizado para o tratamento e exibição dos dados brutos recebidos.

## Como Executar
1. Clone o repositório em sua máquina local.
2. Certifique-se de ter o Python instalado e a biblioteca Requests configurada via pip.
3. Execute o arquivo principal para iniciar a coleta e exibição das cotações.

## Conhecimentos Aplicados
* Desenvolvimento de automações para consumo de dados via API REST.
* Manipulação de dicionários e listas para tratamento de dados JSON.
* Estruturação de scripts voltados para a automação de processos.
